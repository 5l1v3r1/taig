### Fitur
* Bot Like
  * Like all Hastag post
  * Like all User post
  * Like all Popular post
* Bot Komen
  * Comment all Hastag post
  * Comment all User post
  * Comment all Popular post
* Bot Follow
* Bot Unfollow

### Instalasi
```
$ pkg install git python
$ git clone https://github.com/N74NK/taig
$ cd taig
$ pip install -r requirements.txt
$ python taig.py
```
